# Security Policy

## Supported Versions

The following versions of JSM are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.x  | :x:                |

## Reporting a Vulnerability

We take the security of JSM seriously. If you believe you have found a security vulnerability, please do NOT report it via a public issue. 

Instead, please send an email to opensource@bitshepherds.com with a description of the issue. We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.
